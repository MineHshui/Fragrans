# Fragrans
一个纯命令行游戏，运用Visual Studio 2022中的C++进行开发.

通过玩家的操作，通过游戏的所有关卡，即可得到游戏的胜利.

游戏内埋藏有些许彩蛋，需要玩家自行探寻。
#### 操作方法：
        游戏全程都以键盘为操作设备.

        输入选项对应的序号，按下Enter键，即可选择.
        
        选项和输入内容不能输入错误，否则易出现BUG.
